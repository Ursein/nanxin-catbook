## 五、关键约定

### 算法公式

```
Score = α · cos(θ) + β₁ · LikeNorm + β₂ · FollowNorm + β₃ · RatingNorm

cos(θ) = (A · B) / (|A| · |B|)
```

- A, B 为猫咪特征向量（One-Hot 编码）
- 特征来源：毛色标签(colour_tags)、性格标签(personality_tags)、位置(location_area)、性别(gender)
- 热度因子：点赞数(like_count)、关注数(follow_count)、评分均分(avg_rating)
- 权重：α=0.7, β₁=β₂=β₃=0.1（可在配置中调整）
- 应用场景：猫咪详情页"猜你喜欢"推荐 + 搜索结果排序
